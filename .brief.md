# Aichat-Eridian - A multi-target "make" for AI models.

Per the [README](./README.md), AIChat is an all-in-one LLM CLI tool featuring Shell Assistant, CMD & REPL Mode, RAG, AI Tools & Agents, and More.

## Constraints

- this tool is "make" for ai pipelines and applications. see [analysis](docs/analysis/2026-03-02-analysis.md) and [meta-analysis](docs/analysis/2026-03-02-meta-analysis.md)
to understand the tool positioning.
- Use docs/ROADMAP.md as the canonical source of truth for the roadmap. It should point to detailed documentation for each phase (which can change or be renamed)


### Hard

- System is token cost conscious

### Soft

- This tool should function using the "one tool per job" unix ethos
- This tool should use the 'showboat' command to demo its work. Use the output of `showboat --help` to understand how to implement.
- When running `showboat note`, output in a deterministic manner. This is important for `showboat validate` to work.
- Implement both repl and batch interaction surfaces for functionality

### Ask First

- No desktop UI, all Ux work should happen
- Incompatibility with the existing tooling for aichat (argc)
- Reduced compatibility with [llm-functions](https://github.com/sigoden/llm-functions) or [argc](https://github.com/sigoden/argc)
- Introduction of new programming languages
- Significant increase in number of dependencies
